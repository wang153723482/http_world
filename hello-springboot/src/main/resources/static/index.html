http:127.0.0.1:8080/hello

无参数get请求，返回固定的json字符串。

http:127.0.0.1:8080/hello

无参数post请求，返回固定的json字符串，包含session_id。

http:127.0.0.1:8080/helloSession

无参数post请求，返回固定的json字符串，包含session_id。

http://127.0.0.1:8080/hello_body

有body参数post请求，返回参数+时间戳。

http://127.0.0.1:8080/helloPara?name=wangc&age=21

有表单参数post请求，返回请求参数字符串。

http://127.0.0.1:8080/helloParaGet?name=wangc&age=21

有表单参数get请求，返回请求参数字符串。

http://127.0.0.1:8080/helloCookie1

无参get请求，返回cookie。

http://127.0.0.1:8080/helloCookie2

无参get请求，带cookie请求，返回固定字符串。

http://127.0.0.1:8080/helloHeader

无参get请求，返回请求头信息。

http://127.0.0.1:8080/hello_timeout

无参get请求，返回固定的json字符串，2000毫秒才会返回响应。

http://127.0.0.1:8080/hello_timeout

有body参数post请求，返回请求参数+时间戳，5000毫秒才会返回响应。
